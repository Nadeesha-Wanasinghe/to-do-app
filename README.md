# To Do App
This is a simple and efficient way to manage your day-to-day tasks. This application allows users to easily add, delete, and update the status of tasks, helping you stay organized and on top of your priorities. With the seamless integration of the backend API layer and a connected database, your tasks are stored and updated in real-time.

## Features

- ### Add Tasks
  Easily input and save your tasks using the user-friendly interface.

- ### Update Task Status
  Update the status of your tasks as you progress through your day. Mark tasks as completed or pending and stay in control of your to-do list with real-time status changes.

- ### Delete Tasks
  Remove completed or irrelevant tasks effortlessly.

- ### Retrieve all Tasks
  Instant loading of all of your tasks as soon as you connect with the app
